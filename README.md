> **Prerequisites**
>
> - Redwood requires [Node.js](https://nodejs.org/en/) (=20.x) and [Yarn](https://yarnpkg.com/)

Start by installing dependencies:

```
yarn install
```

Then start the development server:

```
yarn redwood dev
```
